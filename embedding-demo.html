Chatbot Embed Options

Five ways to add a chatbot to your site. Try them out, pick what works.

Full Page Modal

Opens centered on the screen with a dark background overlay. Gets full attention.

Works well for: intake forms, consultations, important interactions

Sidebar Panel

Slides in from the right side. Page content stays visible behind it.

Works well for: support while browsing, quick questions, documentation help

Corner Widget

Floating button in the bottom right. Typical chat interface most people recognize.

Works well for: always-on support, help desk, customer service

New Tab

Opens the chat in a separate browser tab. Dead simple to implement.

Works well for: existing buttons, mobile apps, simple links

Inline Embed

Built directly into the page. Good for dedicated contact or help pages.

Works well for: support centers, contact pages, help documentation

Implementation Code

Full Page Modal

HTML

<button onclick="openChat()">Chat</button>

<div id="chat-overlay">
  <div class="modal-box">
    <button class="close-btn" onclick="closeChat()">×</button>
    <iframe src="https://your-chatbot.vercel.app"></iframe>
  </div>
</div>

CSS

#chat-overlay {
  display: none;
  position: fixed;
  inset: 0;
  background: rgba(0, 0, 0, 0.65);
  z-index: 10000;
  align-items: center;
  justify-content: center;
  padding: 20px;
}

#chat-overlay.active { display: flex; }

.modal-box {
  position: relative;
  width: 100%;
  max-width: 920px;
  height: 85vh;
  background: white;
  border-radius: 12px;
  overflow: hidden;
}

.close-btn {
  position: absolute;
  top: 20px;
  right: 20px;
  width: 38px;
  height: 38px;
  background: rgba(0, 0, 0, 0.6);
  border: none;
  border-radius: 50%;
  color: white;
  font-size: 24px;
  cursor: pointer;
  z-index: 1;
}

iframe {
  width: 100%;
  height: 100%;
  border: none;
}

JavaScript

function openChat() {
  document.getElementById('chat-overlay').classList.add('active');
}

function closeChat() {
  document.getElementById('chat-overlay').classList.remove('active');
}

document.addEventListener('keydown', (e) => {
  if (e.key === 'Escape') closeChat();
});

Sidebar Panel

HTML

<button onclick="openSidePanel()">Chat</button>

<div id="chat-sidebar">
  <button class="close-btn" onclick="closeSidePanel()">×</button>
  <iframe src="https://your-chatbot.vercel.app"></iframe>
</div>

CSS

#chat-sidebar {
  position: fixed;
  top: 0;
  right: -480px;
  width: 480px;
  height: 100vh;
  background: white;
  box-shadow: -3px 0 15px rgba(0, 0, 0, 0.1);
  transition: right 0.3s cubic-bezier(0.4, 0, 0.2, 1);
  z-index: 10000;
}

#chat-sidebar.active { right: 0; }

iframe {
  width: 100%;
  height: 100%;
  border: none;
}

JavaScript

function openSidePanel() {
  document.getElementById('chat-sidebar').classList.add('active');
}

function closeSidePanel() {
  document.getElementById('chat-sidebar').classList.remove('active');
}
